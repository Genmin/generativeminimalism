While each expression shares the same basic structure, every one of their features is unique. The SVG path commands are populated with randomly generated numbers within bounds that I set. This structure makes possible trillions of non-repeating variations.

The SVG images above and throughout this page are examples of output from this framework. I invite you to read their code and notice each randomized arc path's granular values.

Process:

I start with concentric circles in Figma, defining their radii and linearly-eased gradients.

Critically, each gradient is slightly asymmetric relative to its container circle.

I then convert each circle to an arc, and randomize the arcâ€™s starting angle and ending angle. I also randomize each gradientâ€™s rotation independent of its arc.

Once the randomization is complete, I comb through each instance and select the most beautiful and expressive for inclusion in this collection. The acceptance rate is about 30% of the total generated.

Roadmap:

launch new generative work in editions of 512 each about twice a month, never using the same fundamental geometry more than once.

Bio:

Joey Roth (b 1984) is an artist and product designer based in San Francisco.

His work includes the Sorapot teapot (2008), Ceramic Speakers (2010), Steel Speaker (2014) and Osma (2021)

Generative Minimalism is his first work on the blockchain.
